# Resources Directory

## Назначение
Вкладка «Ресурсы» содержит источники трафика, инструменты арбитража и аналитику. Часть пунктов может быть «подключаемой», часть — справочной.

## Группы (пример)
### Источники трафика
- TikTok
- Facebook / Meta
- YouTube
- Telegram Ads
- (другие по мере расширения)

### Аналитика / Метрика
- Метрика (внутренняя вкладка после выдачи ключа)
- Трекеры
- Пиксели / события
- CRM/BI (если нужно)

## Карточка ресурса
- Название и иконка
- Статус: not_connected / connected / review / error
- Кнопки: «Подключить» / «Открыть» / «Инфо»
- (опционально) Баланс/лимиты/аккаунт

## Поведение
- До выдачи ключа «Метрика» недоступна
- Для подключаемых ресурсов предусмотрены формы API-key/OAuth (если используется)
