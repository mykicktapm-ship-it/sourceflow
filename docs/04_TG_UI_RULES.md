# Telegram UI Rules

## Общие принципы
- Приложение работает только внутри Telegram WebApp
- Учитывается safe-area Telegram
- Интерфейс рассчитан на узкий viewport
- Управление только через touch

## Навигация
- Используется встроенная кнопка Back Telegram
- Основная навигация через нижнее меню
- Новые экраны открываются через модальные окна

## UX ограничения
- Отсутствие hover-состояний
- Минимальные анимации
- Быстрая обратная связь

## Дополнительно
- Вертикальный скролл
- Без горизонтального скролла
- Haptics используются опционально
