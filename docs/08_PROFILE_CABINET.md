# Profile Cabinet

## Общие данные
- Баланс (TON)
- Привязанный кошелёк
- История транзакций

## Основные действия
- Пополнить баланс (через TON Connect)
- Приобрести тариф (с баланса)
- Активные кампании
- Успешные кампании
- Оставить заявку

## Активные кампании
Отображаются карточки:
- Название
- Статус
- Потрачено
- Прогресс

## Оставить заявку
Форма с полями:
- Ниша
- Гео
- Ссылка на сервис
- Бюджет
- Контакты
- Метрики (CRM, CPI и др.)

Все данные автоматически связываются с Telegram аккаунтом.
