# App Shell

## Структура
Приложение состоит из одного основного контейнера с нижней навигацией.

## Нижняя навигация
- Кабинет
- Услуги
- Создать ордер
- Ресурсы

## Глобальные модалки
- Подключение кошелька
- Оплата
- Фильтры
- Детали ордера
