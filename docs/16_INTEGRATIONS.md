# Integrations

## Типы интеграций
1) Подключаемые (реальные)
- OAuth
- API Key / Token

2) Справочные
- Просто ссылки и инструкции без хранения токенов

## Где хранить токены
Если базы данных нет, токены/ключи не хранятся в приложении.
Возможные режимы:
- ключ вводится пользователем каждый раз при необходимости
- ключ передаётся менеджеру через Telegram Bot (рекомендуемый подход)
- ключ хранится на стороне менеджера/оператора (вне Mini App)

## Права и безопасность
- Mini App не должен светить токены в UI
- Любые секреты не сохраняются в localStorage без крайней необходимости
