# TON Connect

## Назначение
TON Connect используется для подключения TON-кошелька пользователя к Mini App.

## Процесс подключения
1. Пользователь нажимает кнопку подключения
2. Выбирает кошелёк
3. Подтверждает соединение

## Связка данных
- telegramUserId ↔ walletAddress

## Статусы
- not_connected
- connected
- revoked

## Поведение
- Один кошелёк на один Telegram аккаунт
- При отзыве требуется повторное подключение
