# Screens Map

## Основные вкладки
- Кабинет
- Услуги
- Создать ордер
- Ресурсы

## Переходы (ключевые цепочки)
Кабинет → Пополнить → Оплата → Успех  
Кабинет → Активные кампании → Детали ордера/кампании  
Кабинет → Оставить заявку → Отправлено → Ожидание ответа  

Услуги → Детали услуги → Купить/В корзину → Оплата → Pending → Подтверждение менеджера  

Создать ордер → Wizard шаги → Сводка → Оплатить/Оплатить позже → Pending payment (до 48ч) → Paid → Ожидание менеджера  

Ресурсы → Источник → Подключить/Инфо  
(после выдачи ключа) Кабинет/Метрика → Графики/Статистика
