# Services Catalog

## Назначение
Вкладка «Услуги» содержит готовые цифровые услуги, которые пользователь может купить сразу или добавить в корзину.

## Категории (пример)
- Арбитраж / закуп трафика
- Креативы (баннеры, видео, UGC)
- Аналитика и трекинг
- Запуск и оптимизация
- Прогрев / аккаунтинг
- Техническая настройка (пиксели, события, домены)

## Карточка услуги
- Изображение
- Название
- Короткое описание
- Цена
- Сроки выполнения (если применимо)
- CTA: «Купить» / «В корзину» / «Оставить заявку»

## Покупка и обработка
- Оплата с баланса или через TON Connect (если баланс пуст)
- После оплаты: статус «pending» до подтверждения менеджером
- Услуга может быть привязана к кампании (если пользователь укажет это в форме/комментарии)
