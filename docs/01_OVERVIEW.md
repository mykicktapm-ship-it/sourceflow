# EVILS — Overview

EVILS — это современный адаптивный сервис для арбитража и закупки трафика внутри Telegram, созданный для работы без долгих переписок, уточнений условий и ручного менеджмента. Пользователь самостоятельно формирует ордера или выбирает готовые цифровые услуги, управляет бюджетом и ожидаемым трафиком через агрегированные интерфейсы, а команда EVILS подключается только на этапе подтверждения и запуска.

## Цель
Предоставить арбитраж и закуп трафика как сервис (Traffic-as-a-Service) полностью внутри Telegram Mini App.

## Ключевые разделы
- Личный кабинет
- Готовые цифровые услуги
- Создание ордера
- Ресурсы и источники трафика

## Ограничения
- Только Telegram WebApp
- Полный адаптив под Telegram viewport
- Не является обычным веб-сайтом
