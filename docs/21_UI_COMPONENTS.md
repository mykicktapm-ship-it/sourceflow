# UI Components

## Базовые компоненты
- Bottom Navigation
- Header (user avatar, username, wallet status, balance)
- Cards (campaign/order/service)
- Badges (status)
- Tabs (внутри разделов)
- Modals (connect wallet, pay, filters, order details)
- Forms (request/order/service)
- Toasts (success/error/info)
- Progress bar (budget/spent)

## UI States
- loading
- empty
- error
- success
- pending (ожидание оплаты/подтверждения)
